# Portfolio

Un site portfolio professionnel créé avec React et des thèmes de bleu foncé.

## Caractéristiques

- 🎨 Thème bleu foncé moderne et élégant
- 🇫🇷 Interface entièrement en français
- 📱 Responsive design
- ⚛️ Construit avec React 18
- ✨ Animations fluides et interactives

## Installation

```bash
npm install
```

## Démarrage du serveur de développement

```bash
npm start
```

L'application s'ouvrira sur [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build pour la production

```bash
npm run build
```

Cela crée un dossier `build` optimisé pour la production.

## Structure du projet

```
/public          - Fichiers publics (HTML)
/src
  /components    - Composants React
  App.js         - Composant principal
  index.js       - Point d'entrée
  index.css      - Styles globaux
```

## Personnalisation

- Remplacez l'avatar dans `Hero.js`
- Modifiez votre nom et titre professionnel
- Ajustez les couleurs dans les fichiers CSS si nécessaire

Les couleurs principales utilisées :
- Fond sombre: `#0a1f33`
- Bleu primaire: `#1a3a52`
- Bleu accentué: `#4a9eff`
